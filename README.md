## This project only detect image view from the front of person's face and your image size must be 216x216 pixels.
! However this project will provide you a simple code to crop your image that fit requirement. To do this put your raw image to ImageRaw directory then run Image_proccess.py.

### How to use.
1. After finish prepare image run EnccodeGenerator.py to encode image
2. Run main.py for the rest
